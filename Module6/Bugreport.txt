3. В случае выбора услуги в документе Заказ клиента невозможно выбрать количество, поле недоступно для редактирования

4. Шаги для воспроизведения:
	1) Открываем список заказов
	2) Создаем новый заказ
	3) Добавляем новую строку в табличную часть Товары, выбираем номенклутуру с видом "Услуга"
	4) Пробуем ввести количество в строке табличной части

5) Фактический результат
Можно ввести количество в строке товаров

6) Ожидаемый результат
Поле ввода количества недоступно для редактирования
